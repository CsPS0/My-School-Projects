## 🎮 Controls  

### 📜 Menu Navigation  
- 🔢 **Numbers (1-9)** – Navigate  

### 🃏 In-Game Commands  
- 🔢 **Numbers (1-9)** – Play corresponding card  
- 🃏 **D** – Draw a card  
- 🚪 **E** – Exit
